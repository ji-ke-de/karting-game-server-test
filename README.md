# Colyseus Matchmaking System

A simple matchmaking system built with Colyseus for multiplayer games.

## Quick Start

1. Install dependencies:
  
  ```bash
  npm install
  ```

2. Run the server:

  ```bash
  npx ts-node index.ts
  ```
Server runs at `ws://localhost:2567`

3. Run the test script:
  
  ```bash
  npx ts-node test.ts
  ```
